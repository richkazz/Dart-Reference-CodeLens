# Change Log

All notable changes to the "dart-reference-codelens" extension will be documented in this file.

## [1.0.1] - 2024-01-08
- Added support for Enums.
- Added configuration to ignore specific methods.
- Improved caching and performance.
- Added icon.

## [1.0.0] - 2024-01-07
- Initial release.
- Shows reference counts for Classes, Methods, Properties, and Constructors.
- Highlights unused symbols.
